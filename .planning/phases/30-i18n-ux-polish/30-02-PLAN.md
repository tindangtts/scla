3. Update `artifacts/web/public/sw.js` — PRESERVE existing push notification handlers and ADD caching:

   Add at the TOP of the file (before push handlers):
   - CACHE_NAME constant "scla-v1"
   - STATIC_ASSETS array: ["/", "/offline.html", "/manifest.json"]
   - `install` event: pre-cache static assets, call skipWaiting()
   - `activate` event: delete old caches, call clients.claim()
   - `fetch` event: network-first for navigation requests (falling back to /offline.html on network failure), cache-first for static assets (script, style, image destinations) with runtime cache population

   Strategy details:
   - Navigation requests (request.mode === "navigate"): try network first, on failure serve cached /offline.html
   - Static assets (script/style/image): check cache first, if miss then fetch and cache the response clone
   - API requests (anything to /api/): always network, never cache
   - Non-GET requests: ignore (pass through)

   KEEP the existing `push` and `notificationclick` listeners at the bottom of the file.

4. Create `artifacts/web/public/offline.html`:
   - Self-contained HTML file (all styles inline, no external dependencies)
   - Content: "You're offline" heading, "Please check your internet connection and try again." paragraph
   - "Retry" button that calls window.location.reload()
   - Star City Living branding in header bar
   - Inline styles: centered layout, blue (#2563eb) header bar, system font stack
   - Include @media (prefers-color-scheme: dark) for dark background (#0a0a0a) and light text
