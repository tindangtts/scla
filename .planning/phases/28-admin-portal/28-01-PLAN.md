**1. Create `artifacts/web/src/lib/queries/admin-dashboard.ts`:**
Export `getAdminDashboardStats()` that returns `{ totalResidents, totalGuests, openTickets, inProgressTickets, activeBookings, totalRevenue }`.

Queries (all use `db` from `@/lib/db` and schema tables from `@workspace/db/schema`):
- `totalResidents`: COUNT from usersTable WHERE userType = 'resident'
- `totalGuests`: COUNT from usersTable WHERE userType = 'guest'
- `openTickets`: COUNT from ticketsTable WHERE status = 'open'
- `inProgressTickets`: COUNT from ticketsTable WHERE status = 'in_progress'
- `activeBookings`: COUNT from bookingsTable WHERE status = 'upcoming'
- `totalRevenue`: SUM of paidAmount from invoicesTable (coalesce to 0)

Use `sql<number>` for counts cast to `::int`, `sql<string>` for revenue cast to `::text`.

**2. Create `artifacts/web/src/lib/queries/admin-users.ts`:**
Export three functions:
- `getUsers(search?: string, roleFilter?: string)`: SELECT from usersTable with optional ILIKE on name/email and optional eq on userType. Order by createdAt desc. Return `User[]`.
- `getUserById(id: string)`: SELECT from usersTable WHERE id. Return `User | null`.
- `getUserWalletBalance(userId: string)`: SUM wallet transactions (credit - debit) for user. Return string balance.

**3. Rewrite `artifacts/web/src/app/(admin)/dashboard/page.tsx`:**

- Each card uses shadcn Card/CardHeader/CardTitle/CardContent
- Add quick-action links below the grid: "Manage Users" -> /admin/users, "View Tickets" -> /admin/tickets, "View Bookings" -> /admin/facilities
