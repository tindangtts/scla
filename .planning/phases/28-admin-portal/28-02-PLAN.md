**1. Create `artifacts/web/src/lib/queries/admin-tickets.ts`:**
- `getAllTickets(status?: string)`: SELECT all tickets (no userId filter) with optional status filter. Join usersTable to get user name and email. Order by createdAt desc. Return array of ticket + user info.
- `getTicketById(id: string)`: SELECT single ticket by id (no user scope). Join usersTable for submitter info. Return ticket + user or null.
- `getStaffMembers()`: SELECT from staffUsersTable WHERE isActive = true. Return StaffUser[]. Used for assignment dropdown.

**2. Create `artifacts/web/src/lib/queries/admin-facilities.ts`:**
- `getAllFacilities()`: SELECT all from facilitiesTable order by name. Return Facility[].
- `getAllBookings(facilityId?: string, status?: string)`: SELECT all bookings. Join usersTable for booker name. Optional facilityId and status filters. Order by date desc, startTime desc. Return array of booking + user name.

**3. Create `artifacts/web/src/app/(admin)/tickets/page.tsx`:**
- `export const dynamic = "force-dynamic"`, `requireAdmin()`
- Accept searchParams for `status` filter
- Import `getAllTickets`
- Render filter bar: status select (All, Open, In Progress, Completed, Closed) as a GET form
- Render table with columns: Ticket #, Title, Category (capitalize), Status (Badge — open=default, in_progress=secondary with yellow text, completed=secondary with green text, closed=outline), User, Unit, Created
- Each row links to `/admin/tickets/{id}`
- Show total count in header

**4. Create `artifacts/web/src/app/(admin)/tickets/[id]/page.tsx`:**
- `export const dynamic = "force-dynamic"`, `requireAdmin()`
- Import `getTicketById` and `getStaffMembers`
- If not found, `notFound()`
- Render ticket detail:
  - Header: ticket number, title, status Badge, category
  - Detail card: description, service type, unit number, attachment (if attachmentUrl, render as img), created date
  - Submitter card: user name, email
  - Status update form: select dropdown with all 4 statuses, bound to `updateTicketStatus` action
  - Assignment form: select dropdown of active staff members (value=staff.id, label=staff.name), bound to `assignTicket` action. Show "Currently assigned to: {name}" if assignedTo is set.
- Back link to /admin/tickets

**5. Create `artifacts/web/src/app/(admin)/tickets/[id]/actions.ts`:**

  - Link to bookings: "View Bookings" link to `/admin/facilities/bookings?facilityId={id}`
