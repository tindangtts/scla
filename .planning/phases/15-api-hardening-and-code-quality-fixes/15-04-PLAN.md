IMPORTANT — auth-middleware.ts attaches user with a cast:
```typescript
(req as Request & { user: typeof user }).user = user;
```
This means downstream handlers need a typed `AuthenticatedRequest` to access `req.user`.
The executor MUST add this type export to auth-middleware.ts:
```typescript
import { usersTable } from "@workspace/db";
export type AuthenticatedRequest = Request & {
  user: typeof usersTable.$inferSelect;
};
```

Usage pattern in route handlers after migration:
```typescript
// Instead of:
router.get("/", async (req, res) => {
  const payload = requireAuth(req, res);  // OLD: inline helper
  if (!payload) return res.status(401)...;
  // uses payload.userId
})

// Use:
router.get("/", requireAuth, async (req: Request, res: Response) => {
  const { user } = req as AuthenticatedRequest;  // req.user is the full DB user row
  // use user.id instead of payload.userId
})
```
