  - Use drizzle raw sql: `db.select({ balance: sql<string>`COALESCE(SUM(CASE WHEN ${walletTransactionsTable.type} = 'credit' THEN ${walletTransactionsTable.amount} ELSE -${walletTransactionsTable.amount} END), 0)` }).from(walletTransactionsTable).where(and(eq(walletTransactionsTable.userId, user.id), eq(walletTransactionsTable.category, "wallet")))`

- Return: `{ balance: parseFloat(balanceResult), transactions: [...mapped], total, page, limit }`

**GET /deposit** (security deposit balance + transactions):
- Same pattern but filter by `category = "deposit"`
- Return: `{ balance: parseFloat(balanceResult), transactions: [...], total, page, limit }`

**Rewrite POST /invoices/:id/pay** in `artifacts/api-server/src/routes/invoices.ts`:

Replace the current mock payment redirect with real wallet deduction. The endpoint should:

1. Accept body: `{ }` (no payment method needed — always deducts from wallet now)
2. Get the invoice by id + userId (existing pattern)
3. If invoice not found or already paid, return 404/400
4. Compute outstanding = parseFloat(totalAmount) - parseFloat(paidAmount). Use integer cents for arithmetic: `Math.round(parseFloat(totalAmount) * 100) - Math.round(parseFloat(paidAmount) * 100)`, then divide by 100.
5. Compute wallet balance using same SUM query as wallet.ts (category="wallet", userId)
6. If wallet balance < outstanding, return 400 `{ error: "insufficient_balance", message: "Wallet balance insufficient" }`
7. Use `db.transaction(async (tx) => { ... })`:
   - Insert debit transaction: `tx.insert(walletTransactionsTable).values({ userId: user.id, type: "debit", amount: String(outstanding), description: "Bill payment - " + invoice.invoiceNumber, reference: invoice.id, category: "wallet" })`
   - Update invoice: `tx.update(invoicesTable).set({ paidAmount: invoice.totalAmount, status: "paid" }).where(eq(invoicesTable.id, invoice.id))`
8. Return `{ success: true, invoice: { id, invoiceNumber, status: "paid", paidAmount: parseFloat(invoice.totalAmount) } }`

Import walletTransactionsTable from @workspace/db, import sql from drizzle-orm. Import pool from @workspace/db if needed for transaction (check drizzle docs — `db.transaction()` should work with the existing drizzle instance).

**Add POST /admin/wallet/:userId/adjust** in `artifacts/api-server/src/routes/admin.ts`:

Add this endpoint before the export default:

1. requireAdmin check
2. Accept body: `{ amount: number, type: "credit" | "debit", reason: string, category?: "wallet" | "deposit" }`

4. If type is "debit", check current balance (same SUM query, filtered by category) >= amount. If not, return 400 insufficient balance.

6. Call auditLog with action "wallet_adjust", targetType "wallet", targetId req.params.userId, metadata { amount, type, reason, category }
7. Return 201 with the inserted transaction
