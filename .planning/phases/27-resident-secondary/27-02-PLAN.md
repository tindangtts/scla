Create query helpers following established pattern:

**discover.ts:**
- `getAnnouncements()`: Select from announcementsTable where isDraft=false AND type='announcement', order by isPinned desc then publishedAt desc. Limit 20.
- `getNewsletters()`: Select from announcementsTable where isDraft=false AND type='newsletter', order by publishedAt desc. Limit 20.
- `getPromotions()`: Select from promotionsTable where isActive=true AND (validUntil is null OR validUntil >= now()), order by createdAt desc. Limit 20.
- `getAnnouncementById(id: string)`: Select one announcement by ID where isDraft=false.
- `getNewsletterById(id: string)`: Select one newsletter (type='newsletter') by ID where isDraft=false.
- `getPromotionById(id: string)`: Select one promotion by ID.

**info.ts:**
- `getInfoCategories()`: Select all from infoCategoriesTable, order by name.
- `getArticlesByCategory(categoryId: string)`: Select from infoArticlesTable where categoryId matches, order by publishedAt desc.
- `getArticleById(id: string)`: Select one article by ID.
- `getFaqs(category?: string)`: Select from faqsTable where isPublished=true, optionally filter by category, order by sortOrder.
