## Critical Finding: Admin Routing Architecture

The `(admin)` directory is a Next.js route group — parentheses are stripped from URLs. Admin pages are at `/dashboard`, `/tickets`, `/users`, etc., NOT `/admin/dashboard`, etc.

**Impact:**
- All 9 sidebar navigation links point to `/admin/*` paths → 404
- All redirect() calls after mutations go to `/admin/*` → 404
- All revalidatePath() calls target `/admin/*` → stale data
- Middleware `/admin` check only covers `/admin/login`

**Fix:** Rename `(admin)` to `admin` (remove parentheses). Single directory rename fixes all issues.
